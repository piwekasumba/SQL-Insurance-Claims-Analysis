# SQL-Insurance-Claims-Analysis
SQL project analyzing insurance claims data using PostgreSQL.  Includes data cleaning, joins, aggregations, and business insights queries.


This SQL project analyzes insurance claims data using PostgreSQL.

The project demonstrates:

• SQL JOINs
• Aggregations using GROUP BY
• Data filtering and segmentation
• Basic ETL process
• Business insights generation

Key Questions Answered:

1. Which regions have the highest claim costs?
2. Do smokers generate higher claims?
3. Which customers have the highest insurance claims?
4. How do claims vary by age group?
