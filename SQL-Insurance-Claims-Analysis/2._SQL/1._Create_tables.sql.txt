Commit 1 – Project Initialization

This commit marks the start of the SQL Insurance Claims Analysis project.

The goal of this project is to build a structured approach to working with insurance claims data using SQL, while improving understanding of how data behaves across multiple related tables.

At this stage, the following was set up:

- Project folder structure
- Initial README file outlining project scope and direction
- Data folder for organizing datasets used in analysis

The focus moving forward is to:

- Explore claims data using SQL queries
- Work with joins across multiple tables
- Pay attention to how query structure affects results and consistency

• This project is part of my learning process as I build a stronger foundation in SQL and data-related work, with a focus on clarity, structure, and reliability of outputs.
